Complete File Structure
================================================================================

iam-jobs/
│
├── index.html                      [MAIN PAGE]
│   └── The website homepage with job listings table
│
├── README.md                       [DOCUMENTATION]
│   └── Complete project documentation
│
├── DEPLOYMENT.md                   [DEPLOYMENT GUIDE]
│   └── Step-by-step deployment instructions
│
├── QUICKSTART.md                   [QUICK START]
│   └── 5-minute deployment guide
│
├── UPLOAD_GUIDE.md                 [THIS GUIDE]
│   └── Instructions for uploading to GitHub
│
├── requirements.txt                [PYTHON DEPENDENCIES]
│   └── Required Python packages
│
├── .gitignore                      [GIT IGNORE]
│   └── Files to exclude from git
│
├── .github/
│   └── workflows/
│       └── update-jobs.yml         [AUTO-UPDATE CONFIG]
│           └── GitHub Actions workflow (runs every 6 hours)
│
├── css/
│   └── style.css                   [STYLING]
│       └── Professional CSS styling (MET-inspired)
│
├── js/
│   └── app.js                      [FRONTEND LOGIC]
│       └── Search, filter, sort functionality
│
├── data/
│   └── jobs.json                   [JOB DATA]
│       └── Job listings (auto-updated by GitHub Actions)
│
└── scripts/
    └── scrape_jobs.py              [JOB SCRAPER]
        └── Python script to aggregate jobs

KEY FILES TO UPLOAD IN ORDER:
================================================================================

   ✓ index.html
   ✓ README.md

   ✓ requirements.txt

   Settings → Pages → Deploy from main branch

2. Enable GitHub Actions:
